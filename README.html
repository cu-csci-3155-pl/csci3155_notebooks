Readings and In-Class Lecture Notes for CSCI 3155

Sriram Sankaranarayanan (Computer Science, CU Boulder)

This repository contains the jupyter scala notebooks for Principles of Programming: CSCI 3155, University of Colorado Boulder. The repository also contains the jupyter notebooks with examples that were used during lectures.

You can read the notebooks in a couple ways: * In Github - GitHub previews Jupyter notebooks but sometimes Github does not render the mathematical notation correctly. * In JupyterHub - directly viewing the notebooks in JupyterHub (e.g. https://coding.csel.io ) lets you experiment with the examples * As HTML - This is useful when browsing files on your compute
